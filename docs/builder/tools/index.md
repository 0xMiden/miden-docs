# Tools

## CLI

The Miden CLI (`miden-client`) provides a command-line interface for interacting with the Miden network — creating accounts, executing transactions, syncing state, and managing notes. See the [CLI reference](./cli/index.md) for the full command list.

## Additional Tools

- **Playground** - Interactive environment for testing Miden assembly
- **Explorer** - Block explorer for the Miden network
- **[Miden Guardian](../miden-guardian/)** - Server and SDKs for backing up, syncing, and coordinating private account state (built by OpenZeppelin)
