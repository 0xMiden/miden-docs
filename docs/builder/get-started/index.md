## What You'll Learn

The Get Started section covers:

- **Installation** - Install Miden tools using the `midenup` toolchain
- **CLI Basics** - Create your wallet and mint your first tokens
- **Accounts** - Create and manage Miden accounts programmatically
- **Notes & Transactions** - Learn Miden's unique note-based transaction model
- **Your First Smart Contract** - Build, test, and deploy contracts on Miden

Select a released version to access these guides.
